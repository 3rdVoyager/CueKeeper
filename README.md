# CueKeeper

**CueKeeper** is a browser-based line memorization tool for theater actors.  
This is a demo version (v0.1.0) preloaded with *The Two Noble Kinsmen* script.  

## Features in this version
- Select one or more characters if you play multiple roles
- View your current line along with the character name
- Navigate lines with **Next Line** and **Restart**
- Clean and simple interface for easy rehearsal

## How to Use
1. Open `index.html` in your browser
2. Select your character(s) from the dropdown
3. Click **Next Line** to reveal your next line
4. Click **Restart** to reset lines to the beginning
